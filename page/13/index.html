Symbols count in article: 184 Reading time ≈ 1 mins.

zaproxy 又名owasp-zap,不过在最新版kali已没集成所以要手动安装

apt install zaproxy -y 进行安装。

这是一个图形化界面工具，相对来说应该还是挺易用的。

最简单的使用方式:
设置好URL to attack后，点击Attack，开始运行。

如果看到Web Browser XSS Protection Not Enabled说明网站下有易受XSS跨站攻击的页面。

ssh_id_rsa密码破解

Posted on 2020-04-17 Edited on 2021-02-23 In Note , Tools
Symbols count in article: 236 Reading time ≈ 1 mins.

ssh2john下载地址

使用ssh2john解密出私钥密码的hash值.
格式
python ssh2john [hash_file]


使用他这段哈希值用john爆破.

john –wordlist=[wordlist_file] [hash_file]

使用john时要注意不要直接使用/usr/share/wordlists/rockyou.txt.gz，要解压出来成rockyou.txt文件才能使用，否则会报错

百思不得其解的报错，后来才发现不允许直接使用rockyou.txt.gz
