蚁剑修改默认User-Agent

Posted on 2020-09-25 Edited on 2021-02-23 In Note , Tools Views:
Symbols count in article: 323 Reading time ≈ 1 mins.

摸鱼ing

蚁剑不修改User-Agent的话，默认是antSword/v2.1，这实在是太憨批了，这不摆明告诉了别人溯源大手子说：嘿，我用了蚁剑，快来溯源我！

所以还是要做些隐藏手段的，虽然蚁剑能在管理shell中添加自定义的header字段，但是总感觉没有修改客户端参数方便。

修改办法

修改以下两个文件。

request.js

修改第21行的const USER_AGENT

update.js

153行和165行的.set(‘User-Agent’)
